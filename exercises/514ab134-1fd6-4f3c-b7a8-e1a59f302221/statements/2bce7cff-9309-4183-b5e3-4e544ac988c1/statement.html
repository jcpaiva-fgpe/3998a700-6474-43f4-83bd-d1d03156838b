Colecciones

MongoDB almacena documentos en Colecciones.

Las Colecciones son análogas a las tablas en el modelo relacional.

Crear una colección explícitamente

Aunque MongoDB puede crear una colección cuando se inserta el primer documento en la misma, también nos permite crear explícitamente una colección con el comando:

db.createCollection(name, options)

Ejemplo

Para crear la colección prueba, podemos ejecutar la siguiente orden:

db.createCollection("prueba")
