Insertar varios documentos en una instrucción

Evidentemente, si necesitamos insertar varios documentos en una colección, podríamos ejecutar varias sentencias db.collection.insertOne()

No obstante, una mejor solución es insertar todos los documentos en una sola instrucción:

db.collection.insertMany(<array de documentos>)

Ejemplo

Para insertar los datos de varias ventas en la colección de ventas, podríamos ejecutar la siguiente orden:

db.ventas.insertMany(
    [
        {
            "fecha_venta": "2025-01-01",
            "cliente": "Noa Smith",
            "precio_total": 200
        },
        {
            "fecha_venta": "2025-01-02",
            "cliente": "Adam Gustavson",
            "precio_total": 350,
            "descuento": 0.1
        },
])

Podemos observar que la estructura de los documentos es menos rígida que las de las tuplas, ya que cada documento podrá incluir campos distintos al del resto de los registros
