Insertar documentos

Estamos acostumbrados a ver datos en formato tabular, debido a la estructura de la base de datos relacional. Por ello, comenzaremos con los datos en formato tabular, como se ve a continuación, y luego los convertiremos en un objeto JSON para que se pueda insertar en nuestra colección.

fname lname salary departments hiredate
John Doe 70000 sales, admin 2018-08-29

Convirtiendo esta tabla en un objeto, un conjunto de pares campo:valor rodeado por {}se ve así:

    {
	    "fname": "John",
        "lname": "Doe",
        "salary": 70000,
        "departments": ["sales", "admin"],
        "hiredate": "2018-08-29"
    }

Ahora que tenemos nuestro objeto JSON, insértelo en nuestra colección.

Insertar un documento

El comando para insertar un documento en una colección es el siguiente:

db.collection.insertOne()

Ejemplo

Para insertar los datos de una venta en la colección de ventas, podríamos ejecutar la siguiente orden:

db.ventas.insertOne({
    "fecha_venta": "2025-01-01",
    "cliente": "Noa Smith",
    "precio_total": 200
})
