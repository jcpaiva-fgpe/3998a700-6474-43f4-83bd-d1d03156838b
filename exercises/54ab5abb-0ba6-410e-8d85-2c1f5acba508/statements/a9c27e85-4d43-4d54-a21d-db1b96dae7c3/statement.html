Valores diferentes en un campo

Para obtener un recuento de valores únicos en un campo determinado, reemplacecountDocuments() por distinct(), añadiendo un nombre de campo como primer argumento:

db.<collection>.distinct(<campo>)

Ejemplo

Para mostrar los valores diferentes del campo hp en la colección de cards de la base de datos pokemon, podríamos ejecutar la siguiente orden: 

use pokemon
db.cards.distinct("hp")

A partir de ahora, no añadiremos el comando para cambiar de base de datos use <base de datos>

Muestra los valores diferentes del campo rated de la colección movies_2015
