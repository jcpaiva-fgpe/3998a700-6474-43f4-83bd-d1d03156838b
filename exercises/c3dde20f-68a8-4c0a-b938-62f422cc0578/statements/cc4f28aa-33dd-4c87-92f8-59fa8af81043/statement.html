Acciones previas

Para la realización de los siguientes ejercicios, se ha creado una versión reducida de la collección movies que tenéis disponible en la base de datos de sample_mflix. En la máquina virtual, podéis ejecutar MongoDB Compass y conectaros a localhost para acceder a ésta y al resto de bases de datos instaladas.

En el caso de que la colección movies_2015 no aparezca dentro de sample-mflix podéis crear una nueva colección movies_2015 e importar el fichero json, que puedes encontrar en el curso.

Para importar el fichero, selecciona la nueva colección desde MongoDB Compass y haz click en el botón Add Data -> Import JSON or CSV File

Contar documentos

El comando para contar los documentos almacenados en una colección es el siguiente:

db.<collection>.countDocuments(<document>)

Ejemplo

Para contar los documentos almacenados en la colección de ventas, podríamos ejecutar la siguiente orden:

db.ventas.countDocuments()

Ejercicio

Cuenta los documentos existentes en la colección movies_2015
